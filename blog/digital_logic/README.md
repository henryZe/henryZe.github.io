![excess_3](./images/excess_3.png)

3. 格雷码 (gray code)

![grey_code](./images/grey_code.png)

* 任何两位相邻编码只有1位码元不同
* 无权码
* 优势：连续变化时，每次只有1位码元发生改变，比较可靠
